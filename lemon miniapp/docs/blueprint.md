# **App Name**: TokenSpark

## Core Features:

- User Authentication: Secure login/registration using email, Google, Apple, or social media accounts.
- GiftToken Dashboard: Display of GiftToken balance, USDC/USDT balance, and transaction history (earn, spend, transfer, redeem).
- Token Transfer: Send GiftTokens to other users via QR code scan or recipient ID.
- Rewards Catalog: Browse a curated catalog of rewards redeemable with GiftTokens.
- Rewards AI assistant: Use AI to generate recommendations for the best value/relevant rewards from the catalog.
- User Profile: View and manage user profile information, gamified progress, and notification settings.

## Style Guidelines:

- Primary color: Violet (#A06CD5) for a modern, dynamic feel.
- Background color: Light lavender (#F0E6F5), a desaturated version of the primary color.
- Accent color: Electric blue (#7DF9FF) for CTAs and highlights to give a jolt of energy.
- Font: 'Inter', a grotesque-style sans-serif suitable for both headlines and body text, creating a clean, contemporary look.
- Use simple, rounded icons (like lucide-react) to ensure an accessible user experience and reinforce the app's friendly nature.
- Incorporate smooth animations for transactions and achievements, such as confetti or sparkle effects, to communicate positive feedback.
- Design a minimalist layout with rounded buttons to create an accessible, user-friendly interface, following a 9:16 aspect ratio for mobile screens.
